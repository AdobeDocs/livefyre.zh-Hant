# 限制Twitter使用者的內容{#restrict-content-from-twitter-users}

您可以禁止Twitter使用者提取到您的應用程式中。**[!UICONTROL Network Settings > Streams]** 此標籤可讓您禁止「串流」中的使用者，在提供並按一下 **[!UICONTROL Add]**的欄位中輸入其Twitter使用者名稱。

如有需要，請使用核取方塊來刪除所有列出的使用者現有內容。

>[!NOTE]
>
>刪除現有內容無法復原。
