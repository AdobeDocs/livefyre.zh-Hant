# 2018年月日{#march}

2018年月日發行的發行說明。

## 新特性 {#section_syx_mdt_wcb}

此發行版本的生產版本包含下列功能：

* **刪除應用程式。**新增在Studio中刪除Apps的能力，讓使用者更能管理應用程式清單。刪除應用程式會將它從表格移除，但不會從您的網站移除應用程式。應用程式會持續接收來自串流的內容(如果已設定為這麼做)。
