# 功能卡{#feature-card}

將單一精選內容加入您的頁面。

使用功能卡，在單一卡片顯示器上反白顯示個別內容。 應用程式會排定最新「精選」內容的優先順序，並循環檢視最新項目，以確保最新且相關的顯示。

功能卡將顯示最新精選項目（不一定是最近張貼的項目），讓您選擇最佳內容，以便在頁面上即時呈現。
