# 在Comments中內嵌投票{#embed-polls-in-comments}

在您的留言或即時部落格串流中內嵌投票。

1. 在Studio的「App Manager」（應用程式管理員）頁面中，按一下&#x200B;**[!UICONTROL </>]**（內嵌）按鈕，產生民調問答的內嵌代碼。
1. 將內嵌代碼複製並貼至應用程式的&#x200B;**[!UICONTROL Add Custom HTML]**&#x200B;欄位。
