# 移動導覽指南{#move-the-navigation-guide}

指定畫面上導覽指南的位置。

1. 在Storify App **[!UICONTROL Navigation Guide]** Designer中，切換切換。
1. 從 **[!UICONTROL Orientation]** 下拉式清單中選取上方、左側或右側。
