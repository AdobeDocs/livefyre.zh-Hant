# 將翻譯集應用於網路{#apply-a-translation-set-to-a-network}

1. 導覽至Studio中的「設定」。 請確定您是在網路層級工作，而非網站層級工作。
1. 選擇「Network Settings（網路設定）」。
1. 選取「應用程式設定」。
1. 導覽至「預設設定>轉譯集」。
1. 從下拉菜單中選擇一個翻譯集。
