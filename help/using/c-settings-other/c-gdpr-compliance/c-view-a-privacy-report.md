# 檢視隱私權報告{#view-a-privacy-report}

檢視Livefyre中的隱私權報告，以瞭解使用者在Livefyre中記錄的資訊。

1. **[!UICONTROL Settings > Privacy]**前往。
1. 從清單中按一下隱私權報告以檢視該報表。
