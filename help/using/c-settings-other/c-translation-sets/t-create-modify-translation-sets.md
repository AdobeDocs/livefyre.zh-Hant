# 建立和修改翻譯集{#create-and-modify-translation-sets}

1. 導覽至Studio中的「設定」。
1. 選取「網路設定」。
1. 選取翻譯集。
1. 按一下+設定以建立新的翻譯集。
1. 為您的設定指定名稱和主要語言。
1. 修改您要顯示在應用程式名稱底下的文字。例如，若要變更「注釋應用程式」所使用字串的文字，請修改「注釋」下方的欄位。
1. 儲存翻譯集。
