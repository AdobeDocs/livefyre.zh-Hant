# Sides Bookmarklet{#sidenotes-bookmarklet}

Sideras可載入至（幾乎）任何現有頁面。

若要示範，請使用下列書籤小程式：

Siestroms書籤小程式

只需將書籤小程式複製到書籤中，然後按一下連結即可載入Siteras。 系統會提示您輸入選擇器，以用來裝飾目前頁面上的Sidestras。
