# 從應用程式選取內容以加入功能{#select-content-to-feature-from-an-app}

直接從應用程式功能或取消功能內容。

1. 從應用程式中，將滑鼠指標暫留在您要加入的內容上。
1. 按一 **[!UICONTROL Feature]** 下以加入內容，或 **[!UICONTROL Unfeature]** 停止加入內容。
