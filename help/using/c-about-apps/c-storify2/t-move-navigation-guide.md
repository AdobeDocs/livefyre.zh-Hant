# 移動導覽指南{#move-the-navigation-guide}

在畫面上指定導覽指南的位置。

1. 在Storify 2 App Designer中，開啟&#x200B;**[!UICONTROL Navigation Guide]**&#x200B;切換。
1. 從&#x200B;**[!UICONTROL Orientation]**&#x200B;下拉式清單中選取top、left或right。
