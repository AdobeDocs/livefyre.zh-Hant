# 趨勢分析{#trending}

使用趨勢分析來展示特定網站上最活躍的系列、推動網站流量進入您最活躍的討論，或讓讀者一目瞭然地評估網站活動。 應用程式會顯示從最活躍到最不活躍的「系列」頂端，並會顯示一個橫條圖，顯示每個「系列」的相對人氣。 趨勢分析可協助讀者判斷最多對話的發生地，並引導他們進行這些對話。
