# 聊天室{#chat}

在您的網站上啟用即時聊天。

聊天功能可讓觀眾參與即時活動、公告或節目的即時對話。 內容會以連續串流的無執行緒聊天方式顯示，以利快速參與並活躍頁面上的活動。

福克斯新聞利用聊天捕捉到了觀眾對貝爾格里奧當選教皇的看法和反思。 在宣佈消息後幾分鐘內，Fox新聞頻道就啟動了聊天，以便立即進行對話。
