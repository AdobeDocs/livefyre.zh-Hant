# 地圖功能{#map-features}

地圖應用程式為您的網站提供這些功能。



* 內容即時輸入地圖。
* 來自相同區域的內容會進行叢集，並可在「圖庫」模式中檢視。
* 放大「地圖」，以專注特定區域的內容。
