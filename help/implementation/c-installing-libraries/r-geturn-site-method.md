# Geturn網站方法{#geturn-site-method}

此方法會傳回此網站的URN。

## Java範例 {#section_nyl_ycs_rz}

使用方式：

```
site.getUrn();
```

輸出範例：

```
"urn:livefyre:network=`example.fyre.co`:site=1" 
```
