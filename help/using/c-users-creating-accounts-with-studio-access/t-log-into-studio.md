# 登入Studio{#log-into-studio}

當您的網路帳戶連線至Studio帳戶後，就可以透過您的書簽連結或任何Livefyre應用程式存取Studio。

1. 從應用程式中，從 **[!UICONTROL Studio]** 您的 **[!UICONTROL Account]** 下拉式清單中選取。
1. 如果您的工作階段過期，Livefyre會將您重新導向至Studio中的Studio登入或Studio。
