# 將翻譯集應用於站點{#apply-a-translation-set-to-a-site}

**將翻譯集應用於站點**

1. 導覽至Studio中的「設定」。
1. 選取網站。
1. 選取「網站設定」。
1. 選取「應用程式設定」。
1. 導覽至「預設設定>轉譯集」。
1. 從下拉菜單中選擇一個翻譯集。
