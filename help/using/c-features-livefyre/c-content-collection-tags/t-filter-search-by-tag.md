# 依系列或內容標籤篩選搜尋{#filter-search-by-collection-or-content-tag}

依「系列」標籤篩選Studio中的內容搜尋。

1. 在面 **[!UICONTROL Tag]** 板的搜尋下拉式清單中 **[!UICONTROL Collections]** 選取。
1. 輸入搜尋的標籤，或按一下「系列」 **[!UICONTROL Tag]** 下方所列的。

   清單中會顯示所有具有選取之標籤的系列。
