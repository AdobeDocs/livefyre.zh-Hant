# 將影像新增至民調問答{#add-images-to-a-poll}

新增影像，為民調問答增添視覺興趣。

1. 按一下問題或答案右側的圖示。
1. 將影像拖放至提供的視窗，或使用Filepicker從Instagram、Flickr、Dropbox或列出的其他提供者中選取影像。
1. 視需要裁切影像。
1. 按一下&#x200B;**[!UICONTROL Save]**&#x200B;將影像新增至民調問答。
1. 將滑鼠指標暫留在影像上，以檢視問題或選擇，然後按一下顯示的圖示，以裁切、刪除或取代影像。
