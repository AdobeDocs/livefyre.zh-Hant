# 使用精選API匯總精選內容{#aggregated-featured-content-using-the-featured-apis}

Livefyre提供四個精選API，讓您擷取輸入網路、網站或系列的50個最新精選內容項目，或輸入系列頂端的兩個內容部分。

## 特色(網路) {#section_cgm_1nw_xz}

擷取輸入網路的50個最新精選內容(及其作者)。(使用分頁來傳回所有內容，而非僅限最近50個。)

**資源**

```
GET https://{networkName}.bootstrap.fyre.co/bs3/v3.1/{network}/featured 
```
