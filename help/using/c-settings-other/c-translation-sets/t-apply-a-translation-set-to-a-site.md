# 將翻譯套用至網站{#apply-a-translation-set-to-a-site}

**將翻譯套用至網站**

1. 導覽至Studio中的「設定」。
1. 選取網站。
1. 選取「網站設定」。
1. 選取「應用程式設定」。
1. 導覽至「預設設定>翻譯集」。
1. 從下拉式選單中選取「翻譯集」。
