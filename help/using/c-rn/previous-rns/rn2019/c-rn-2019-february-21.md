# 發行說明- 2019年2月21日{#rn022119}

**2019年2月21日更新的發行說明。**


## 錯誤修正

* 修正重新整理 Instagram 商業帳號權杖時發生的「帳號重複」錯誤。
* 修正使用自訂指令碼更新應用程式時，發生重複建立應用程式的問題。
* 修正 URL 搜尋所顯示 Instagram 內容會在搜尋結果中正常顯示縮圖，但無法在「檢視詳細資訊」或發佈至應用程式時正常顯示的問題。
* 修正按一下「載入更多」結果時部分產品 ID 發生錯誤的問題。實施流量分類。
