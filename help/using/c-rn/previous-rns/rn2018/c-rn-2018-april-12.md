# 2018 年 4 月 12 日{#april}

2018年4月12日發行說明。

## 新功能 {#section_syx_mdt_wcb}

以下是此版本生產版本的新功能：

您現在可以為串流中的智慧標籤選擇信賴臨界值。 設定標籤的精確度分數(0-100)可讓您控制我們擷取的資產的精確度。
