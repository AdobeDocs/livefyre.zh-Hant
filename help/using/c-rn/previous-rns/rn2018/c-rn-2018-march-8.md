# March 8, 2018{#march}

2018年3月8日發行說明。

## 新功能 {#section_syx_mdt_wcb}

以下是此版本生產版本的新功能：

* **刪除應用程式。 **新增在Studio中刪除應用程式的功能，讓使用者可以更好地管理應用程式清單。 刪除應用程式會從表格中移除它，但不會從您的網站移除應用程式。 如果應用程式設定為從串流接收內容，它將會繼續。

## 問題 {#section_ehw_ndt_wcb}

本版本已解決下清單格中的問題。

| **問題類型** | **元件** | **發行說明** |
|---|---|---|
