# 發行說明-2019年月21日 {#rn022119}

**2019年月21日更新版本的發行說明。**


## 錯誤修正

* 修正重新整理Instagram業務帳戶Token造成「重復帳戶」錯誤的問題。
* 修正使用自訂指令檔更新應用程式時，會發生錯誤的應用程式複製錯誤的問題。
* 修正從URL搜尋顯示Instagram內容時，會在搜尋結果中正確顯示縮圖，但不在「檢視詳細資料」中或發佈至應用程式的問題。
* 修正按一下「載入更多」會導致部分產品ID發生錯誤的問題。
