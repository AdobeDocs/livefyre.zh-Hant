# 登入Studio{#log-into-studio}

在您的網路帳戶連線至Studio帳戶後，您就可以透過書籤連結或任何Livefyre應用程式存取Studio。

1. 從應用程式中，從下拉式 **[!UICONTROL Studio]** 清單中 **[!UICONTROL Account]** 選取。
1. Livefyre會將您重新導向至Studio登入頁面，或在作業過期時重新導向至Studio。
