# 將站點添加到網路{#add-a-site-to-a-network}

您可以為共用相同使用者設定檔的網域和子網域，將新網站新增至網路。

您可在Studio中新增和顯示多達1,200個網站。 網站會依字母順序列出。

要向網路添加新站點，請執行以下操作：

1. 按一下 **[!UICONTROL Network]** Studio選單列中的項目，以開啟 **[!UICONTROL Select Network or Site]** 面板。
1. 使用搜尋欄位來搜尋網站，或按一下 **[!UICONTROL Add new site]** 以新增網站。

   此面板列出了選定網路中包含的所有站點。

1. 在提供的 **[!UICONTROL Name]** 字 **[!UICONTROL URL]** 段中輸入和，然後按一下 **[!UICONTROL Add Site]**。
