# 從應用程式選取內容至功能{#select-content-to-feature-from-an-app}

直接從應用程式功能或取消功能內容。

1. 在應用程式內，將滑鼠指標暫留在您想要的內容上。
1. 按一下 **[!UICONTROL Feature]** 以功能或 **[!UICONTROL Unfeature]** 停止裝飾該內容。
