# buildLivefyreToken網路方法{#buildlivefyretoken-network-method}

傳回加密的有效Livefyre Token，可用來與其所呼叫之網路的其他Livefyre API互動。

傳回加密的有效Livefyre Token，可用來與其所呼叫之網路的其他Livefyre API互動。

依預設，此Token會設為在建立時的24小時內過期。

```
network.buildLivefyreToken();
```
