# Geturn Network方法{#geturn-network-method}

此方法會傳回此網路的URN。

## Java範例 {#section_nyl_ycs_rz}

```
network.getUrn(); 
```

輸出範例：

```
"urn:livefyre:network=`example.fyre.co`" 
```
