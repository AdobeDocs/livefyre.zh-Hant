# 趨勢分析{#trending}

使用趨勢來展示特定網站上最有效的「系列」、推動網站流量的進行，或允許讀者評估網站活動。應用程式會顯示從最多到最不活動的排名最前的「精選」，並顯示顯示每個系列相對人氣的橫條圖。趨勢分析可協助讀者判斷最多對話的位置，並將它們導向這些對話。
