# 新增貼文至導覽指南{#add-posts-to-the-navigation-guide}

將貼文新增至Storify 2的導覽指南。

1. 在Storify 2中撰寫貼文。
1. 撰寫貼文的頁首。 在新增頁首之前，您無法顯示貼文的導覽反白顯示。
1. 按一下貼文上的&#x200B;**[!UICONTROL Navigation Guide]**&#x200B;圖示，將其新增至導覽指南。
1. 確認您要將貼文新增至導覽指南。
