# getNetworkName Network Method{#getnetworkname-network-method}

此方法返回沒有fyre.co的網路名稱。

## Java示例{#section_nyl_ycs_rz}

```
network.getNetworkName();
```

輸出範例：

```
"example" 
```

## NodeJS示例{#section_xkd_gds_rz}

```
network.getNetworkName();
```

輸出範例：

```
"example" 
```

```
$network->getNetworkName(); 
```

