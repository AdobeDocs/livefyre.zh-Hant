# 依系列或內容標籤篩選搜尋{#filter-search-by-collection-or-content-tag}

依「系列」標籤篩選Studio中的內容搜尋。

1. 在&#x200B;**[!UICONTROL Collections]**&#x200B;面板的搜索下拉式清單中選擇&#x200B;**[!UICONTROL Tag]**。
1. 輸入搜尋的標籤，或按一下「系列」下方所列的&#x200B;**[!UICONTROL Tag]**。

   清單中會顯示所有具有選取之標籤的系列。
