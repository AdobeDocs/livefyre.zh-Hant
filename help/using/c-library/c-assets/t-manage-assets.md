# 檢視資產詳細資料{#view-asset-details}

資料庫可讓您新增資訊至儲存的內容，讓您根據品牌策略組織資產。

1. 將滑鼠移至個別內容片段，並使用顯示在窗格中的按鈕， **[!UICONTROL View Details]** 或管理個別項目。
1. 選取個別或多個內容片段
1. 使用頁面頂端的按鈕來管理內容。
