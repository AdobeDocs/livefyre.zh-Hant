# 從應用程式選取要加入功能的內容{#select-content-to-feature-from-an-app}

直接從應用程式功能或取消功能內容。

1. 從應用程式中，將滑鼠指標暫留在您要加入的內容上。
1. 按一下&#x200B;**[!UICONTROL Feature]**&#x200B;以呈現內容，或按一下&#x200B;**[!UICONTROL Unfeature]**&#x200B;以停止呈現該內容。
