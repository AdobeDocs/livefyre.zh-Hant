# 自訂 HTML{#custom-html}

允許協調者、擁有者和部落格將自訂HTML附件新增至透過「留言」或「即時部落格」發佈的內容。

「自訂HTML內嵌」功能可讓協調者、擁有者和部落格將自訂HTML附件新增至透過「留言」或「即時部落格」發佈的內容。 除了標準影像、視訊播放器和其他由Livefyre應用程式自動擴充的內嵌影像外，此功能還可用來附加任何內容至您的貼文，包括專屬視訊播放器或投票Widget。

請洽詢您的客戶經理，為您的網路或網站啟用此功能。
