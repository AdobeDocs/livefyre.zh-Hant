# 移動導覽指南{#move-the-navigation-guide}

在畫面上指定導覽指南的位置。

1. 在Storify 2 App Designer中，開啟切換 **[!UICONTROL Navigation Guide]** 功能。
1. 從下拉式清單中選取頂端、左 **[!UICONTROL Orientation]** 側或右側。
