# 將翻譯集套用至網路{#apply-a-translation-set-to-a-network}

1. 導覽至Studio中的「設定」。確定您在「網路」層級工作，而不是「網站」層級。
1. 選取「網路設定」。
1. 選取「應用程式設定」。
1. 導覽至「預設設定>翻譯集」。
1. 從下拉式選單中選取「翻譯集」。
