# 在注釋中內嵌投票{#embed-polls-in-comments}

在您的留言或即時部落格串流中內嵌投票。

1. 在Studio的App manager頁面中，按一下 **[!UICONTROL </>]** （內嵌）按鈕以產生民調問答的內嵌代碼。
1. 將內嵌代碼複製並貼至應 **[!UICONTROL Add Custom HTML]** 用程式的欄位。
