# 民調問答{#polls}

讓客戶即時表達意見。

民調問答可讓您針對網站建立並發佈快速、單一問題的民調問答。使用民調問答來鼓勵讀者參與，收集網站訪客的意見回饋、意見和要求。

>[!NOTE]
>
>民調問答不會連結至串流。
