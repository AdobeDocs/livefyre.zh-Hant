# 在導覽指南中設定貼文數目{#set-the-number-of-posts-in-the-navigation-guide}

設定訪客在導覽指南中可看到多少貼文。

1. 在Storify App **[!UICONTROL Navigation Guide]** Designer中，切換切換。
1. 在 **[!UICONTROL Max Number of Guide Posts]** 欄位中輸入數字。如果您未設定此欄位，Livefyre將預設上限為10。
