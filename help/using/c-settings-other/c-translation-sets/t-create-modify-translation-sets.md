# 建立和修改翻譯集{#create-and-modify-translation-sets}

1. 導覽至Studio中的「設定」。
1. 選擇「Network Settings（網路設定）」。
1. 選擇轉換集。
1. 按一下+Set以建立新的翻譯集。
1. 為您的集指定名稱和主要語言。
1. 修改您要在應用程式名稱下顯示的文字。 例如，若要變更「注釋應用程式」所使用字串的文字，請修改「注釋」下的欄位。
1. 保存翻譯集。
