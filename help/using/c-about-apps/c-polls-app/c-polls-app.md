# 民調{#polls}

讓客戶即時發表意見。

民調問答可讓您建立並發佈快速、單題的民調問答至您的網站。 使用民調問答來鼓勵讀者參與、收集來自網站訪客的意見、意見和要求。

>[!NOTE]
>
>民調問答不會連結至串流。
