# 檢視隱私權報告{#view-a-privacy-report}

在Livefyre中檢視隱私權報表，以檢視Livefyre中為使用者記錄的資訊。

1. 前往 **[!UICONTROL Settings > Privacy]**.
1. 按一下清單中的隱私權報告以加以檢視。
