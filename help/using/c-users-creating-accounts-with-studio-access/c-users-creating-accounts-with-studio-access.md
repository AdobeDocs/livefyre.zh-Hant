# 使用者：建立用戶帳戶{#users-creating-user-accounts}

Studio帳戶可讓您存取您擁有權限的所有Studio元件，包括應用程式建立、內容協調和使用者管理。

Studio使用者可存取多個網路，因此可從單一帳戶管理這些網路。

Studio帳戶可讓您存取Livefyre Studio，並與與Livefyre應用程式互動的使用者帳戶不同。
