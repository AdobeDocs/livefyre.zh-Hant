# 依URL搜尋{#search-by-url}

依URL搜尋要儲存在資產庫的內容。

從網頁上的任何URL新增內容。 只要輸入您要在頁面上內嵌的URL，然後按一下「傳回」即可。

傳回的資訊會顯示頁面的資訊卡摘要，包括完整的URL、頁面的標題、影像、署名，以及第一段落的片段。

>[!NOTE]
>
>對於使用URL欄位的Instagram搜尋，作者會傳回為&quot;Instagram&quot;。
