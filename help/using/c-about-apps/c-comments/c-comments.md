# 注釋{#comments}

將即時注釋內嵌至您的應用程式。

「留言」會透過即時對話取代您的預設留言。我們的社交整合功能可讓您輕鬆地跨Twitter和Facebook擷取所有有關您貼文的對話，並將使用者的朋友納入對話中。

Mashable使用注釋來提高使用者參與討論的參與度。功能包括多媒體上傳、社交分享、聆聽器計數和單一登入協助，讓對話變得輕鬆有趣。
