# Sitents Bookmarklet{#sidenotes-bookmarklet}

Sitents可載入(幾乎)任何現有頁面上。

若要示範Sitents使用以下書簽小工具：

Sitents Bookmarklet

只要將書簽小工具複製到書簽中，然後按一下連結即可載入SiteNotes。系統將提示您輸入選擇器，以便在目前頁面上裝飾Sitdones。
