# getUrn網路方法{#geturn-network-method}

此方法返回此網路的URN。

## Java示例 {#section_nyl_ycs_rz}

```
network.getUrn(); 
```

輸出範例：

```
"urn:livefyre:network=`example.fyre.co`" 
```
