# 2018年月12日{#april}

2018年月12日發行的發行說明。

## 新特性 {#section_syx_mdt_wcb}

此發行版本的生產版本包含下列功能：

您現在可以在串流中選擇智慧標記的信賴臨界值。設定標籤的精確度分數(0-100)可讓您控制正在擷取之資產的準確性。
