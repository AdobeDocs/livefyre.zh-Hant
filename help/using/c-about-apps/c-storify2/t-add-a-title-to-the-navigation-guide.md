# 新增導覽指南標題{#add-a-title-to-the-navigation-guide}

為您的導覽指南提供標題，讓網站訪客清楚明瞭。

1. 開啓Storify故事。
1. 在Storify App **[!UICONTROL Navigation Guide]** Designer中，切換切換。
1. 在欄位中輸入標題，填入「新增標題」。」
