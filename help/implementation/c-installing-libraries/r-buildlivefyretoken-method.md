# BuildLiveLifeToken網路方法{#buildlivefyretoken-network-method}

傳回加密有效的Livefyre Token，可用來與來自所呼叫網路的其他Livefyre API互動。

傳回加密有效的Livefyre Token，可用來與來自所呼叫網路的其他Livefyre API互動。

根據預設，此Token會在自建立時間起24小時內過期。

```
network.buildLivefyreToken();
```
