# 發佈內文{#publish-a-story}

將Storify發佈至您的網站。

在變更您的應用程式時，Studio會自動儲存您的應用程式。

>[!NOTE]
>
>按一下 **[!UICONTROL Publish]** 會產生應用程式的內嵌代碼，並更新先前發佈之應用程式的所有現有例項。

1. 按一下 **[!UICONTROL Publish]** 以產生內嵌代碼，或使用目前版本更新此應用程式的任何先前發佈版本。
1. 將內嵌代碼新增至您的頁面，以新增應用程式並顯示任何先前發佈的內容。
